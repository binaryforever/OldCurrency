# Old Currency

A historical currency rate lookup app for iPhone, built with SwiftUI.

## What it does

Old Currency lets you look up the exchange rate between any two currencies on any date going back to 1999. Pick your currencies, pick a date, and tap Submit. That's it.

It's designed for anyone who needs to check what a rate actually was on a specific day — whether that's for travel expenses, accounting, personal finance, or just curiosity.

## Features

- Look up exchange rates for any date from 1 January 1999 to today
- 31 currencies supported, sourced from the European Central Bank and other official institutions
- Step backwards and forwards through dates one day at a time
- Quickly swap the From and To currencies for the reverse rate
- If no rate is available for the selected date (weekends and public holidays), the nearest available rate is shown with a clear note
- Rates rounded to 4 decimal places

## Data source

Rates are provided by the [Frankfurter API](https://frankfurter.dev), an open-source currency data API aggregating official daily exchange rates from central banks and financial institutions worldwide. No API key required.

## Privacy

Old Currency collects no personal data. No tracking, no analytics, no advertising. See the full [privacy policy](https://binaryforever.github.io/OldCurrency/privacy.html).
